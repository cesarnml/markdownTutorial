# What is Markdown?
* Lightweight markup language
* Can be converted to HTML/XHTML
* Highly readable and easy to use

## What can you do with Markdown?
* Headings
* Lists
* Emphasis
* Links
* Blocks of Code
* Images
* Blockquotes
* Horizontal Rule
